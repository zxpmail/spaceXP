# spaceXP

#### 介绍
包装redisTemplate常用功能并增加redis常用配置类

#### 软件架构
包装redisTemplate常用功能并增加redis常用配置类

#### 安装教程
直接生成jar包
#### 使用说明
    <dependency>
        <groupId>cn.piesat</groupId>
        <artifactId>spaceXP-spring-boot-starter-redis</artifactId>
        <version>2.0.0</version>
    </dependency>

    在Service中直接注入
    RedisService redisService即可
